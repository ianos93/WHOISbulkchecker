BulkWHOIS v1.0

Check domain registration status and EPP flags in bulk — any number of domains, processed in batches

Getting Started
  1. Paste domains in the text area — one per line, or space/comma separated
  2. Or drag & drop a .txt or .csv file (domain column is auto-detected)
  3. Click Check Domains — large lists are processed in batches of 50 with automatic pausing between batches to respect rate limits
  4. Filter by status or EPP flag, then Export CSV if needed. Hit Cancel at any time to stop mid-run
Results Explained
  • Available Domain returned no RDAP record (404) — likely unregistered
  • Registered Domain has an active RDAP record with registration data
  • Error Lookup failed — registry may not support RDAP, or timed out
  • cached Result loaded from your local 24-hour cache — no API call made
EPP Status Flags
  • clientHold / serverHold Domain is suspended — DNS not resolving
  • pendingDelete Queued for deletion — may drop soon
  • pendingTransfer Transfer to another registrar in progress
  • ok / active Domain is live with no restrictions
  • Hover any EPP tag in the results table for a plain-English explanation
Technical Notes
  • Uses the IANA RDAP bootstrap to query each TLD's authoritative registry directly, with rdap.org as a proxy fallback
  • Rate limited to 5 requests / 2 seconds. Large lists are split into batches of 50 with a 4-second pause between batches to avoid overwhelming registries
  • Results are cached in localStorage for 24 hours — re-checking known domains is instant and skips the rate limiter entirely
  • All lookups run entirely in your browser — no data is sent to any server
  • RDAP coverage is best for .com, .net, .org and modern gTLDs. Some ccTLDs may return errors
