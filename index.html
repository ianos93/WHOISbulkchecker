Quick Instructions

  • Enter card names (one per line) or upload a .txt decklist file
  • Select image quality and options (Paper Only, First Print)
  • Click "Summon Cards" to fetch from Scryfall
  • Download individual images as ZIP or visual decklist as PNG

Formats Supported

  • Card names: Lightning Bolt
  • With quantities: 4 Lightning Bolt or 4x Lightning Bolt
  • With set codes: Lightning Bolt (M10) or Lightning Bolt M10
  • With collector numbers: Savannah Lions DMR 270 or Lightning Bolt M10 #341
  • Deck sections: Deck, Sideboard, //Sideboard, Commander
Tip: Use set codes + collector numbers to fetch specific printings (e.g., Sol Ring 6ED 350)

What's New in v3.0

  • Print for Proxy — generate a print-ready PDF at true card size (63×88mm)
  • A4 & A3 support with 10mm safe margins for home and shop printers
  • Proxy page preview with full navigation before downloading
  • Repeat by Quantity and Ignore Basic Lands proxy options
  • Visual Decklist improvements — sideboard column with label
  • Smoother UI transitions and emoji-free interface

Cache Management

Cards are cached for 7 days to speed up repeated fetches. Clear the cache if you need to re-fetch specific printings.
