WHO'SWHO

RDAP

Bulk domain registration checker — entirely client-side, privacy-first

What is this?

WHO'SWHO checks domain registration status using the RDAP protocol (Registration Data Access Protocol) — the modern successor to WHOIS. Everything runs in your browser; no data is sent to any server except the official RDAP endpoints.

How to use

  • Paste one domain per line in the text area, or drag and drop a .txt or .csv file
  • Click Check Domains to start the lookup
  • Results show availability, EPP status codes, registrar, expiry, and more
  • Export to CSV when done

Privacy & Caching

Results are cached in your browser's local storage for 24 hours to avoid redundant lookups. No external analytics or tracking. Clear the cache anytime using the button below the input area.

EPP Status Codes

EPP codes indicate a domain's state:

  • clientTransferProhibited — transfer to another registrar is locked
  • clientUpdateProhibited — domain settings can't be changed
  • serverHold — domain is suspended
  • pendingDelete — domain is scheduled for deletion

…and many more. You can filter results by EPP code after checking domains.

Rate Limiting

To avoid overwhelming RDAP servers, WHO'SWHO processes domains in batches with automatic pauses. The speed depends on how many domains you're checking and the responsiveness of each TLD's RDAP server.
